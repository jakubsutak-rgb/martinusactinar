📚 Martinusáčtinár — návod na spustenie

Krok za krokom, ako keby si mal 6 rokov. Žiadne technické skúsenosti nepotrebuješ.

Čo ťa čaká:
① Získaš prístupový kód k AI (API kľúč) — trvá ~5 minút
② Nasadíš tool na Vercel (bezplatný hosting) — trvá ~5 minút
③ Prepojíš jedno s druhým — trvá ~2 minúty
④ Zdieľaš link kolegom — hotovo!
1
Získaj API kľúč (prístupový kód k AI)
Toto je ako elektrická prípojka — bez nej tool nefunguje. Zaregistruješ sa a zaplatíš malý kredit.
1
Otvor stránku Anthropic Console
V prehliadači (Chrome, Safari, Firefox — hocičom) otvor adresu:
https://console.anthropic.com
Anthropic je firma, ktorá vytvára AI model Claude — ten istý, ktorý generuje texty v našom tooli.
2
Klikni na "Sign up"
Na stránke uvidíš toto:
Welcome to Claude
Sign in to your account or create a new one
Continue with email
— or —
Continue with Google
Odporúčam kliknúť Continue with Google a použiť firemný Google účet. Je to najrýchlejšie.
3
Dokonči registráciu
Zadaj meno, prijmi podmienky (klikni "I agree") a potvrď e-mail ak príde správa. Po registrácii ťa presmeruje na Dashboard — hlavnú stránku.
Ak vidíš nadpis "Anthropic Console" a v ľavom menu sú položky ako "API Keys", "Workspaces" — si na správnom mieste!
4
Dobij kredit (min. $20)
Bez kreditu API kľúč nefunguje. V ľavom menu klikni na Billing, potom Add credits.
Billing → Credits
Current balance: $0.00
Add credits
Zadaj čiastku $20 a platobnú kartu. Toto je jednorazový kredit — neopakuje sa automaticky.
Cena za 1 generovaný text (≈ 500 slov)~$0.003
$20 kredit vydrží cca6 000 textov
Pre tím 10 ľudí to stačí naniekoľko mesiacov
Kredit sa míňa len keď tool skutočne niekto používa. Kým nič negenerujete, neklesá.
5
Vytvor API kľúč
V ľavom menu klikni na API Keys. Uvidíš takúto stránku:
API Keys
No API keys yet. Create one to get started.
+ Create Key
Klikni Create Key. Objaví sa okno — do poľa "Name" napíš napríklad martinusactinar a klikni Create Key.
6
Skopíruj a ulož kľúč — DÔLEŽITÉ!
Po kliknutí sa objaví kľúč. Vyzerá takto:
sk-ant-api03-xXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxXxX
Tento kľúč sa zobrazí IBA RAZ. Skopíruj ho a ulož do bezpečného miesta (napr. do poznámkového bloku alebo 1Password). Ak ho zatvoriš bez skopírovania, musíš vytvoriť nový.
Klikni na ikonu kopírovania vedľa kľúča, potom Done.
API kľúč máš! Teraz ideme na hosting.
2
Nasaď tool na Vercel
Vercel je bezplatný hosting. Vezme kód z GitHubu a spraví z neho živú webstránku.
1
Zlúč vetvu do main na GitHube
Otvor repozitár jakubsutak-rgb/martinusactinar na GitHub.com. Uvidíš žltý banner:
claude/hopeful-darwin-4btey0 had recent pushes Compare & pull request
Klikni Compare & pull request → potom Create pull request → potom Merge pull request → Confirm merge.
Kód je teraz v hlavnej vetve (main). Môžeme nasadiť.
2
Otvor Vercel
V novej záložke prehliadača otvor:
https://vercel.com
3
Zaregistruj sa cez GitHub
Na úvodnej stránke Vercelu klikni Sign Up, potom vyber:
▶ Continue with GitHub
Prihlásiš sa rovnakým GitHub účtom, kde je repozitár martinusactinar. GitHub sa opýta "Authorize Vercel" — klikni Authorize.
Registrácia na Verceli je zadarmo. Netreba kartu.
4
Pridaj nový projekt
Po prihlásení uvidíš Dashboard. Klikni na veľké tlačidlo:
+ Add New… → Project
5
Vyber repozitár martinusactinar
Vercel zobrazí zoznam tvojich GitHub repozitárov. Nájdi martinusactinar a klikni Import vedľa neho.
📁 jakubsutak-rgb/martinusactinar
Import
Ak repozitár nevidíš, klikni "Adjust GitHub App Permissions" a povol prístup k tomuto repozitáru.
6
Konfiguruj projekt — zatiaľ NIČ nemeň
Objaví sa stránka "Configure Project". Všetky nastavenia nechaj tak ako sú. Teraz NEKLIKAJ "Deploy" — najprv treba pridať API kľúč!
Ak klikneš Deploy teraz bez API kľúča, tool sa síce nasadí, ale nebude generovať texty. Pokračuj krokom 7.
7
Pridaj API kľúč do Environment Variables
Na tej istej stránke "Configure Project" rozklikni sekciu Environment Variables (klikni na jej nadpis). Zobrazí sa formulár:
Environment Variables
Key
ANTHROPIC_API_KEY
Value
sk-ant-api03-xXxXx… (sem vlož tvoj kľúč)
Add
Do poľa Key napíš presne toto (veľkými písmenami, pomlčky):
ANTHROPIC_API_KEY
Do poľa Value vlož API kľúč, ktorý si skopíroval v Časti 1 (začína na sk-ant-).
Klikni Add a uvidíš, že kľúč sa pridal do zoznamu (hodnota bude skrytá hviezdičkami).
API kľúč je bezpečne uložený na serveri. Používatelia toolu ho nikdy neuvidia.
8
Klikni Deploy!
Teraz klikni na veľké modré tlačidlo:
🚀 Deploy
Vercel začne buildovať a nasadzovať tool. Uvidíš animovaný progress. Trvá to asi 60–90 sekúnd.
9
Hotovo — dostaneš link!
Keď nasadenie skončí, Vercel zobrazí konfetové animácie 🎉 a link:
🎉
Congratulations!
Your project has been successfully deployed.
martinusactinar.vercel.app
Tool je živý! Otvor link a skús sa prihlásiť tvojím @martinus.sk e-mailom.
3
Zdieľaj s kolegmi
Stačí poslať jeden link. Nikto nič neinštaluje.
1
Skopíruj link a pošli ho
Link z Vercelu (napr. martinusactinar.vercel.app) pošli kolegom cez Slack, Teams alebo e-mail.
Link môžeš neskôr zmeniť na vlastnú doménu, napr. copy.martinus.sk — ak to budeš chcieť, daj vedieť a poradím ako.
2
Kolegovia sa prihlásia svojím @martinus.sk e-mailom
Na úvodnej stránke zadajú firemný e-mail (napr. jana.novakova@martinus.sk) a kliknú "Vstúpiť do nástroja". Tool overí, že ide o @martinus.sk adresu — iné neprepustí.
Heslo sa nezadáva — tool overuje len doménu. Ak chcete v budúcnosti silnejšie overenie (napr. skutočné prihlásenie cez Google Workspace), daj vedieť.
3
Kolegovia môžu hneď generovať texty
Vyberú formát (blog, banner, story…), zadajú tému a zadanie, kliknú Generovať v martinusáčtine — a o pár sekúnd majú text.
Tool je hotový a v prevádzke. Netreba nič inštalovať, funguje v akomkoľvek prehliadači na počítači aj mobile.

Q
Koľko to bude stáť mesačne?
Záleží od intenzity používania. Orientačne:
Hosting (Vercel)$0 / mesiac
AI model (Claude Haiku)~$0.003 / text
Pri 500 textoch / mesiac~$1.50 / mesiac
Pri 2 000 textoch / mesiac~$6 / mesiac
Kredit $20 vydrží tímu pravdepodobne niekoľko mesiacov. Keď sa minie, jednoducho dobiješ ďalší.
Q
Tool prestáva fungovať — čo sa stalo?
Najčastejšie príčiny:
1. Minul sa kredit — skontroluj na console.anthropic.com → Billing
2. API kľúč vypršal — vytvor nový v console.anthropic.com → API Keys a pridaj ho do Vercelu (Settings → Environment Variables → zmeň hodnotu → Redeploy)
Q
Chcem vlastnú doménu (napr. copy.martinus.sk)
V Verceli: Settings → Domains → Add Domain → zadaj copy.martinus.sk. Vercel ti dá DNS záznamy, ktoré musí správca domény pridať do DNS nastavení Martinusu. Daj vedieť a pomôžem.
Q
Chcem zmeniť alebo vylepšiť tool
Napíš mi čo chceš zmeniť — upravím kód, pushnem do GitHub a Vercel nasadí zmeny automaticky do ~1 minúty. Žiadna ďalšia práca z tvojej strany.
So želaním dobrej knihy (a funkčného toolu) 📚
